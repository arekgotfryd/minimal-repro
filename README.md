# Hono minimal project

This is a minimal project with [Hono](https://github.com/honojs/hono/) for Cloudflare Workers.

## Prerequisites
- node v20.3.0
- npm v9.6.7

## Features

- Minimal
- TypeScript
- Wrangler to develop and deploy.
- [Jest](https://jestjs.io/ja/) for testing.

## Usage

Install

```
yarn install
```

Develop

```
yarn dev
```

Test

```
yarn test
```

Deploy

```
yarn deploy
```
